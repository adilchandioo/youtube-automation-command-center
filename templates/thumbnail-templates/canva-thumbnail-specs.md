# Canva Thumbnail Specifications & Design System

## YouTube Thumbnail Dimensions
- **Resolution**: 1280 × 720 px
- **Aspect ratio**: 16:9
- **File format**: PNG (high quality) or JPG (lower file size)
- **Max file size**: 2 MB
- **Safe zone**: Keep all important text & faces within center 80% (YouTube crops edges on some devices)

## Setting Up a Canva Template (do this ONCE per channel)

1. Open Canva → Create design → Custom size → 1280 × 720 px
2. Duplicate design once per channel
3. Set up LAYERS (bottom to top):
   - Layer 0: Background (gradient dark/brand color)
   - Layer 1: B-roll / AI generated background image
   - Layer 2: Subject/face (from AI/stock photo)
   - Layer 3: Shape elements (arrows, circles, rectangles)
   - Layer 4: Text (MAX 3 WORDS)
   - Layer 5: Logo watermark (small, bottom-right corner)
4. Save as TEMPLATE in Canva ("CH-001-Thumbnail-Template")
5. Duplicate the template for each new video

## Default Text Style
- **Font**: Bebas Neue (free in Canva) OR Impact (universal bold font)
- **Size**: 130–200 pt for main word, 80–110 pt for secondary
- **Color**: White or bright accent color (yellow, red, neon green)
- **Outline**: 5–8 px thick black outline/shadow on all text
- **Case**: ALL CAPS

## Face/Subject Style
- Position: slightly left or right of center (not dead center — leaves room for text)
- Size: subject face takes 40-60% of canvas height
- Emotion: extreme (shock, amazement, confusion, excitement) — NEVER neutral
- Lighting: bright, high contrast, eyes well-lit

## Color Presets (per channel — create brand kit in Canva)

- Text: white (#FFFFFF)
- Text outline: black (#000000)

## 5 Thumbnail Layout Templates (reusable)

### Layout 1: Face + Side Text (most reliable)
```
[          ] [BIG     ]
[          ] [TEXT    ]
[  FACE    ] [HERE    ]
[          ] [➡️       ]
[          ] [         ]
```
- Face right, left side text OR face left, right side text
- Arrow/red circle pointing to face or text
- Use for: list videos, how-tos, personal stories

### Layout 2: Split Screen (before/after, comparison)
```
[ BEFORE  ][ AFTER   ]
[ (image) ][ (image) ]
[          ][         ]
[ BIG NUM ][ BIG NUM ]
```
- Left side labeled "BEFORE" / "0" / "POOR", right labeled "AFTER" / "$X" / "RICH"
- Use for: transformation, case study, comparison videos

### Layout 3: Big Object + Arrow
```
[    👀     ]
[           ]
[  OBJECT   ]
[  (big)    ]
[   🔴→     ]
```
- Central object (money stack, iPhone, AI robot, screenshot)
- Red circle/arrow pointing at important detail
- 1-2 word text above
- Use for: product reviews, tutorials, tools, "this app" videos

### Layout 4: Question / Shock (text-focused)
```
[           ]
[  WHY DID  ]
[  THEY DO  ]
[   THIS?   ]
[ face: ??? ]
```
- Big provocative question filling 60% of screen
- Small confused/shocked face in corner
- Use for: controversy, "you won't believe", mystery titles

### Layout 5: Number Dominance
```
[  $10,000  ]
[           ]
[  /month   ]
[  👉 face  ]
```
- Giant number (earnings, percent, stat) filling 50% of screen
- Face pointing/reacting to the number
- Use for: earnings videos, stats, "I made X amount" videos

## Quality Check Before Export

- [ ] Zoom out to 10% — still readable?
- [ ] Compare to last 3 videos on channel — same brand feel?
- [ ] Compare to competitors — does this stand out (not blend in)?
- [ ] Text 3 words or less?
- [ ] Face emotion matches title?
- [ ] No spoilers (thumbnail doesn't give away the ending)?
- [ ] No typos? (critical — you can't edit after upload without re-uploading)
- [ ] File size <2MB?
- [ ] Export as PNG for crisp text, JPG for smaller size
