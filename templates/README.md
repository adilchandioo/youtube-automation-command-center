# 📋 Templates

Fill-in-the-blank templates for every recurring asset. Copy, fill, ship.

## Structure
- `script-templates/`
  - `long-form-template.md` — main 8-15 min video script skeleton
  - `shorts-template.md` — 30-55 sec Shorts/Reels/TikTok script skeleton
- `thumbnail-templates/`
  - `canva-thumbnail-specs.md` — Canva setup + 5 reusable layout templates
- `description-templates/`
  - `long-form-description.txt` — video description with all sections
  - `channel-about-template.txt` — channel About page + channel keywords
- `shorts-templates/`
  - `hook-swipe-file.md` — 50 proven Shorts hooks
- `emails-templates/` (create later) — outreach emails, sponsor pitches, team comms

## Usage

1. Pick the template for the asset you need
2. Copy into your tool (Google Docs, Canva, YouTube Studio)
3. Replace all `{{VARIABLES}}`
4. Customize for the specific video/channel
5. Delete unused sections (e.g., remove affiliate section if no affiliates)
6. Save final version to channel's Google Drive folder

Pro tip: create a **Canva Brand Kit** for each channel so colors/fonts/logos auto-apply to thumbnail templates.
