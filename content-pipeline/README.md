# 🔄 Content Pipeline — 7 Stages From Idea to Publish

Every video goes through these 7 stages in order. Each stage has:
- Clear owner
- SLA (time limit)
- Inputs & outputs
- Quality checklist

Cards move left → right on your Trello/Notion board.

```
💡 Idea → 🔬 Research → ✍️ Script → ✅ Approve Script → 🎙 VO → 🎬 Edit → 🖼 Thumbnail → 📤 Upload → 🚀 Schedule → ✅ Publish
```

---

## Stage Folders in this Directory

| Stage | Folder | Owner |
|---|---|---|
| 1. Idea Generation | `idea-generation/` | Founder / automation |
| 2. Research | `research/` | Writer |
| 3. Scripting | `scripting/` | Writer |
| 4. Voiceover | `voiceover/` | VA (or founder at MVP) |
| 5. Editing | `editing/` | Editor |
| 6. Thumbnails | `thumbnails/` | Designer |
| 7. Publishing | `publishing/` | Uploader VA / founder |

---

## Board Layout (Trello/Notion)

Create these lists:
1. 💡 Idea Backlog (unsorted ideas)
2. 🔬 Researching (actively researching)
3. ✍️ Scripting (writing)
4. ✅ Script Approval (waiting for manager)
5. 🎙 Voiceover (VO being recorded/generated)
6. 🎬 Editing (editor working)
7. 🖼 Thumbnails (designer working)
8. 📤 Ready to Upload (fully rendered, waiting for upload)
9. 🚀 Scheduled (scheduled in YouTube Studio)
10. ✅ Published (live — archive weekly)
11. ❌ Killed (rejected ideas — keep for post-mortems)

---

## Service Level Agreements (SLAs)

| Stage | Time target (MVP) | Scale target |
|---|---|---|
| Idea → Research | Batch weekly | Batch weekly |
| Research → Script draft | 6 hours | 4 hours |
| Script → Approval | 12 hours | 6 hours |
| Approval → VO | 4 hours | 2 hours |
| VO → Edit delivered | 24 hours | 12 hours |
| Edit + Thumbnails → Ready | 12 hours | 6 hours |
| Ready → Scheduled | 4 hours | 2 hours |
| **Total end-to-end** | **~48 hours** | **~24 hours** |

At MVP (you doing most work): aim for 1 video per week per channel.
At scale (full team): 3 long-form/week/channel + daily Shorts.

---

## Bottleneck Detection

Any card stuck in a stage >2x SLA = bottleneck.
- If stuck at "Script Approval" → founder is bottleneck, batch reviews
- If stuck at "Editing" → editor overloaded, hire 2nd or improve SOP
- If stuck at "Thumbnails" → designer slow, use Canva templates to speed up
