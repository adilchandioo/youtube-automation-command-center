# 📤 Stage 7: Publishing & Distribution

Goal: Get the video LIVE on YouTube (and repurposed across Shorts/TikTok/Reels) correctly, on schedule, fully optimized.
Owner: Uploader VA / Founder
SLA: 4 hours from "Ready to Upload" → "Scheduled"

Reference: `sops/SOP-001-DAILY-PUBLISH.md`

---

## Pre-Upload Checklist (before opening YouTube Studio)

- [ ] Final MP4 exported, <128GB (YouTube limit)
- [ ] Thumbnails A & B ready
- [ ] Title(s) chosen from script metadata (1 primary, we A/B titles later with TubeBuddy)
- [ ] Description filled from template
- [ ] Tags ready (15 tags)
- [ ] End screen target videos decided
- [ ] Playlist assignment known
- [ ] Captions burnt in OR SRT file ready
- [ ] Affiliate disclaimer + niche disclaimer present
- [ ] Video watched in full for QA (first 60 seconds + scrub through — see SOP-001)

---

## Upload Steps (YouTube Studio)

1. YouTube Studio → Create → Upload videos
2. Drag MP4 into uploader
3. **DETAILS** tab:
   - Title (chosen title, <70 chars, keyword front)
   - Description (paste final description with all sections filled)
   - Thumbnail: upload Variant A
   - Playlist: add to correct playlist
   - Tags: paste tags
   - Audience: "Not made for kids" (unless it is)
   - Paid promotion: CHECK if any affiliate/sponsor content that needs disclosure
   - Category: choose (Howto & Style, Education, Entertainment, People & Blogs, etc.)
   - Language & captions: English, upload captions if using SRT (auto-captions are OK if burned-in captions exist)
   - Recording date/location: leave blank
   - License: Standard YouTube License
   - Allow embedding: YES (helps with sharing)
   - Shorts option: NO (for long-form; YES for Shorts)

4. **VIDEO ELEMENTS** tab:
   - Add end screen: 2 videos (1 most-relevant, 1 popular + subscribe button)
   - Add cards: 1 at 60% video pointing to related video, 1 at 80% pointing to playlist

5. **CHECKS** tab: wait for copyright check — if claim appears, resolve BEFORE scheduling (see SOP-006)

6. **VISIBILITY** tab:
   - Set to "Schedule"
   - Pick publish date/time (per channel schedule — usually audience peak hour, check analytics)
   - Set to Premiere? Optional (only for big videos / milestone episodes)
   - Save/Schedule

7. After scheduling, copy the VIDEO URL for cross-promotion.

## Shorts Upload (separate)

For each Short clipped from this video:
- Vertical 1080x1920, <60 sec
- Title: ALL CAPS or question, <60 chars
- Description: 1 sentence + "Watch full video: [long-form URL]" + 3-5 hashtags including #shorts
- Visibility: schedule 30 min BEFORE long-form (funnels Shorts viewers to long-form)
- NO end screen/cards
- Audience: not made for kids
- Tag #shorts

---

## Cross-Promotion & Distribution (after it's scheduled)

After scheduling long-form AND Shorts, queue cross-posts:

### Day of publish:
- [ ] Community post (if 1k+ subs): "New video in 3 hours — it's about [tease]. Drop a 🔥 if you're ready."
- [ ] Twitter/X: 30 min before go-live → "New video in 30 minutes — [title + link]"
- [ ] Instagram story teaser: screenshot of thumbnail + swipe up/link sticker
- [ ] Discord/Telegram: ping community when live
- [ ] Email list: send to list when live (if newsletter exists)

### 15 min after publish:
- [ ] Pinned comment (see SOP-001 for template)
- [ ] Reply to first 20-30 comments (heart, reply, engage)
- [ ] Share in relevant subreddits (only if genuinely valuable — don't spam)

### 24-48 hours after publish:
- [ ] Check TubeBuddy A/B thumbnail test — swap to B if running A/B
- [ ] Clip 1 more Short if a specific moment is getting high retention
- [ ] Repost best Short on TikTok & Instagram Reels
