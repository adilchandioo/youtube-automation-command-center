# 👥 Team & People Operations

Building a team is how you stop trading time for money. Hire slow, fire fast, document everything.

---

## Team Structure

### Phase 1 (MVP, Months 1-2)
- **You (Founder)**: Everything — strategy, scripting, VO, editing, thumbnails, upload, analytics
- Goal: get 10 videos out to learn the process yourself before hiring

### Phase 2 (Months 2-4, ~$500/mo revenue)
- **Founder**: Strategy, script approval, weekly review
- **Video Editor** (1, $400–600/mo retainer or $10–15/video): edits + captions
- **Thumbnail Designer** (1, $3–5/thumb, Fiverr/Upwork per-piece): thumbnails
- You still write scripts & upload

### Phase 3 (Months 4-6, ~$3k/mo revenue)
- Add **Script Writer** (1, $8–15/script)
- Add **VA Uploader/SEO** (1, $3–5/hr, part-time)
- Add **Shorts editor** (1, $3–5/Short)
- Founder: strategy, approvals, sponsor outreach, hiring

### Phase 4 (Months 6-12, $10k+/mo)
- **Channel Manager** (1, $800–1,500/mo): runs day-to-day
- 2nd video editor
- Sponsor outreach VA
- Community manager / comment moderator
- Multiple writers/designers across channels
