# Role: Thumbnail Designer

**Reports to**: Founder / Channel Manager
**Rate**: $3–$8 per thumbnail set (2 variants per video)
**Hours**: Per-project freelance (5–20 thumbnails per week)
**Tools**: Canva, Midjourney/DALL-E (we give access if needed), Photoshop (optional)

---

## What You'll Do
- Create 2 thumbnail variants per video (A/B test)
- Create 1 thumbnail per Short
- Maintain consistent brand style across channel
- Use AI image generation for faces/subjects
- Iterate quickly on feedback
- Help refresh template when CTR drops

## Required Skills
- Canva fluency (template-based workflows)
- Experience designing YouTube thumbnails (portfolio required)
- Eye for CONTRAST, COMPOSITION, and STOPPING POWER
- Understand what makes people CLICK
- Can generate AI images with Midjourney/DALL-E (prompt templates provided)
- Typography instincts (size, hierarchy, readability)
- Speed (turnaround <24h per set)

## KPIs
| Metric | Target |
|---|---|
| Avg CTR on your thumbnails | >8% |
| Sets delivered per week | 5–20 |
| Revision requests per set | <1 |
| On-time delivery | >95% |

## Style Guide Per Channel
We provide:
- Brand color codes
- Fonts
- 3 reference thumbnails we like
- Thumbnail templates in Canva

Your job: duplicate template, swap face/object/text, keep brand consistent, make it CLICKABLE.

## Apply
Send portfolio of 5+ YouTube thumbnails. Start with paid test ($8 for 2 thumbnails).
