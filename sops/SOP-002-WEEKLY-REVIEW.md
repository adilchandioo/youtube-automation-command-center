# SOP-002: Weekly Review Ritual

**Version**: 1.0
**Owner**: Founder / Channel Manager
**Frequency**: Every Sunday, 60–90 min
**Goal**: Reflect on KPIs, double down on winners, cut losers, plan next week's slate.

---

## Pre-Review Prep (automated)

Before the meeting, ensure these are updated:
- [ ] `analytics/reports/weekly-report-{date}.md` auto-generated by analytics script
- [ ] KPI tracker Google Sheet synced (latest 7 days of data)
- [ ] Trello/Notion board cleaned: all "Published" cards archived, all stuck cards flagged
- [ ] Competitor digest emailed (top 10 videos from your 10 tracked competitors)

---

## Review Template

Go through every live channel one by one.

### 1. Channel Health Check (per channel)

| Metric | Last Week | This Week | Target | Verdict |
|---|---|---|---|---|
| Subs gained | | | | |
| Total views | | | | |
| Watch time (hrs) | | | | |
| Avg CTR (last 10 videos) | | | >7% | |
| Avg AVD (last 10 videos) | | | >40% | |
| Revenue | | | | |
| Videos published | | | 3+ | |

### 2. Top 3 Videos This Week

For each winner:
- **Title**:
- **Views / CTR / AVD**:
- **Why it won?** (topic? hook? thumbnail? share on Reddit?)
- **Action**: Make 3 more videos on this exact sub-topic within 14 days

### 3. Bottom 3 Videos This Week

For each loser:
- **Title**:
- **Views / CTR / AVD**:
- **Why it flopped?** (bad hook? oversaturated topic? bad thumbnail?)
- **Action**: Kill? Retitle/re-thumbnail? Or simply note and don't repeat the formula

### 4. Backlog Grooming (30 min)

- [ ] Review Idea Backlog — remove anything no longer relevant
- [ ] Add 10 new ideas inspired by this week's winners
- [ ] Assign next week's videos (3 long-form + 14 Shorts per channel)
- [ ] Confirm each next-week video has writer/editor/thumbnail assigned with deadline

### 5. Team Feedback (15 min)

- [ ] Review VA deliverables from week — any quality issues?
- [ ] Send Slack/Discord message praising 1 thing, correcting 1 thing (radical candor)
- [ ] Any late deliveries? Address now (don't let slippage become habit)

### 6. Experiment Log

Pick 1–2 experiments for next week. Examples:
- [ ] New hook formula (e.g., story-first instead of number-first)
- [ ] New thumbnail style (face-only vs text-only)
- [ ] Different upload time (test 7am vs 7pm)
- [ ] New AI voice
- [ ] Longer videos (15 min vs 10 min)

Log experiments in `analytics/reports/experiments-log.md`.

### 7. Financial Check

- [ ] Record this week's revenue (AdSense + affiliates + sponsors)
- [ ] Pay any outstanding VA/contractor invoices
- [ ] Check tool subscriptions — cancel anything not being used
- [ ] Update `BUSINESS_PLAN.md` projections if reality diverges >20%

### 8. Next Week's Commitments (write these down!)

- Number of videos to publish: ___
- Revenue target: $___
- Key experiment: __________
- One bottleneck to fix: __________

---

## Outputs (after review)

1. Updated Trello board — next 7 days of videos assigned + scheduled
2. Weekly report pushed to repo: `analytics/reports/weekly-report-YYYY-MM-DD.md`
3. Team feedback messages sent
4. Dashboard data refreshed
5. Founder's personal to-do list for the week (the 3 things that move the needle)

---

## Red Flags to Escalate Immediately

- CTR dropping 2+ weeks in a row → thumbnail/title crisis, schedule thumbnail redesign
- AVD dropping → scripts are getting formulaic, re-read top winners, freshen up
- Subscriber loss 3+ days straight → something is alienating audience, check recent comments
- Any strike/warning → legal triage, pause uploads if needed
- Revenue drops >30% vs 4-week avg → check RPM, might be seasonal, but investigate
